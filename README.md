# Primer-Proyecto-UiPath

#Descripcion
Este Proyecto de automatizacion Desarrollado por UiPath demuestra la implementacion de arquitectura modular mediante la invocacion de flujos de trabajo (.xaml). El objetivo principal es  mostrar el paso de datos entre flujos utilizando argumentos de entrada (In-Arguments) y variables locales.

#Caracteristicas Tecnicas
-Modularidad: Separacion de Logica en archivos independientes (Main.xaml y Saludador.xaml).
-Paso de Parametros: Uso de 'invoke workflow' para pasar informacion del flujo principal al subproceso
=Manejo de Datos: Uso de variables de tipo String y argumentos con direccion 'In'.

#Herramientas
-UiPath
-.Net Framework
